This is a fanmade project inspired by Hollow Knight and Hollow Knight: Silksong.

The music for this project was created by Christopher Larkin, Strawberry Flavored Demon and GoatFromTheStars.
Please check out their channels and support them if you like their work!

The game was made using the godot engine for a coding assignment for Lyceum Kralingen.

All code was made by us and is available on github.

This is in no way an official game and is not affiliated with Team Cherry in any way.
We do not intend to make any profit from this project and it is purely for educational and entertainment purposes.
